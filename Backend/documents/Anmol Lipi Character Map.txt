**Punctuation & Symbols:**

*   `` ` `` (Backtick) -> ੱ (Adhak - consonant doubler sign)
*   `-` (Hyphen) -> - (Hyphen)
*   `=` (Equals) -> = (Equals)
*   `[` (Opening Bracket) -> । (Dandi - Gurmukhi full stop)
*   `]` (Closing Bracket) -> ।। (Double Dandi - Gurmukhi paragraph end)
*   `\` (Backslash) -> ਞ (Njanja - nasal consonant)
*   `;` (Semicolon) -> ; (Semicolon)
*   `'` (Apostrophe) -> ' (Apostrophe)
*   `,` (Comma) -> , (Comma)
*   `.` (Period/Dot) -> . (Period/Dot)
*   `/` (Forward Slash) -> / (Forward Slash)

**Shift + Punctuation & Symbols:**

*   `~` (Tilde / Shift+Backtick) -> ੱ (Adhak version 2 - consonant doubler sign)
*   `_` (Underscore / Shift+Hyphen) -> - (Hyphen version 2)
*   `+` (Plus / Shift+Equals) -> + (Plus sign)
*   `{` (Opening Brace / Shift+[) -> { (Opening Brace)
*   `}` (Closing Brace / Shift+]) -> } (Closing Brace)
*   `|` (Pipe / Shift+Backslash) -> ਙ (Nganga - nasal consonant)
*   `:` (Colon / Shift+Semicolon) -> : (Colon)
*   `"` (Double Quote / Shift+Apostrophe) -> " (Double Quote)
*   `<` (Less Than / Shift+Comma) -> ੴ  (Ek Onkar)
*   `>` (Greater Than / Shift+Period) -> `Khanda` (Khanda)
*   `?` (Question Mark / Shift+Forward Slash) -> ? (Question Mark)

**Whitespace:**

*   `Spacebar` -> (Space)
*   `Tab` -> (Tab)
